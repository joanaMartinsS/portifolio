Sobre

Trabalho na interseção entre dados e gestão de projetos — e é justamente aí que consigo gerar mais valor. No Instituto Phomenta, faço a gestão de projetos que conectam empresas a ONGs. Tenho facilidade com Python, SQL, Power BI, Looker Studio e Google Sheets. Aprendo rápido e gosto de trabalhar em projetos onde os dados fazem diferença de verdade.

Painel de Projetos de Carbono Florestal

IDESAM · Power BI · Georreferenciamento

Painel interativo que mapeia e analisa projetos de carbono florestal no mercado voluntário brasileiro, com dados do setor AFOLU (Agricultura, Florestas e Uso do Solo). Permite filtrar por região, status, tipo de certificação e estimar créditos de carbono (VCUs) entre 2020 e 2050.

  • Apoio na lógica de georreferenciamento dos projetos no mapa

Limpeza e Integração de Bases Municipais — APA

Área de Preservação Ambiental · Python · Power BI

Projeto de tratamento e integração de mais de 10 bases de dados municipais de uma Área de Preservação Ambiental específica. O resultado final será um painel em Power BI com mapa interativo e buscador, permitindo consultar dados ambientais por município de forma acessível.

O que já foi feito
  • Limpeza e padronização de 10+ bases com Python (pandas)
  • Tratamento de inconsistências, valores nulos e normalização de campos
  • Estruturação do modelo de dados para integração no Power BI

Dashboard de Monitoramento Institucional

Casa Fluminense · Looker Studio · Google Sheets

Projeto de arquitetura de dados e visualização para monitoramento da saúde financeira e de captação de uma organização do terceiro setor. Arquitetura da base de dados
  • 15+ abas organizadas por tema: financeiro, captação, comunicação, programático
  • Abas auxiliares e dicionário de dados para padronização
  • Estrutura preparada para conectar diretamente ao Looker Studio
